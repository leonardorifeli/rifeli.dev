## Variação 3 — leitor curioso / orquestrador

Em 30 dias enviei 1.024 prompts para o Claude Code. Recebi 12.824 respostas.

Mais de 12 respostas para cada coisa que digito.

Não é chat. Cada prompt vira tool calls em cascata: leitura de arquivo, escrita, busca, decisão, execução de comando, novo prompt para si mesmo. Bash domina como ferramenta, seguido de Read e Edit.

Claude Code não é interface de IA. É orquestrador. Quem trata como "ChatGPT na linha de comando" está usando 10 por cento da ferramenta.
